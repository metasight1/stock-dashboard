💡 작동 방식: GitHub Actions가 매주 월/목 한국시간 오전 7시에 Python 스크립트를 자동 실행하여 data.json을 갱신하고, GitHub Pages가 이 페이지를 다시 게시합니다. 데이터 출처는 Yahoo Finance(yfinance)입니다.

🟢 배당주 필터 기준

  • 배당락일이 오늘 + 7일 ~ 오늘 + 14일 사이
  • 배당률: Bull ≥ 2.5% / Mixed ≥ 2.75% / Bear ≥ 3.0%
  • Yahoo Finance recommendationMean ≤ 2.5 (Buy 이상에 해당)

🎯 추천주 필터 기준

  • Yahoo Finance recommendationMean ≤ 2.0 (Buy 이상)
  • 향후 12개월 EPS 성장률 ≥ 10%
  • PEG ≤ 1.5 (가급적)
  • ※ "주간 5회 이상 언급" 조건은 무료 뉴스 API 한계로 V1에서는 제외

🚫 제외 종목

GOOGL, GOOG, AAPL, META, MSFT, AMZN, NVDA, TSLA

⚠️ 데이터 한계

  • Yahoo Finance 무료 데이터를 사용하며, Finviz/TipRanks/Morningstar의 컨센서스와는 차이가 있을 수 있습니다.
  • 추천주의 "주간 언급 횟수"는 무료 API로는 정확히 측정이 어려워 V1에서는 제외됩니다. V2에서 NewsAPI 등 추가 가능.
  • 실제 매수 전 반드시 본인이 증권사 앱에서 수치를 재확인하세요.
